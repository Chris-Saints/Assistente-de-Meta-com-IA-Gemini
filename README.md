<p>O <strong>Assistente de IA, Gemini</strong>, é um projeto de um curso básico introdutório da RocketSeat sobre a implementação de IA nos projetos. Ensinando como utilizar a Gemini API, criando uma KEY e botando em prática como efetivamente colocar em um projeto de forma funcional. O projeto em questão se trata de um assistente para tirar dúvidas em jogos. No caso 3 (Valorant, League of Legends e Counter Strike). Fazendo uma pergunta, escolhendo o jogo e clicando em perguntar, você receberá uma resposta.</p>

<h1>Segurança</h1>

<p>Algo importante que eu aprendi nesse curso é a importancia da sua KEY para as API's, porém no curso eles explicam isso, mas não demonstram nenhuma forma de proteger essa informação de qualquer usuário.<br/>
Portando tomei a <strong>iniciativa</strong> de pesquisar e estudar formas para assegurar que minha KEY fique protegida e que mesmo assim o projeto funcione normalmente. Decidi portanto fazer um <strong>Serverless</strong>, usando a <strong>Vercel</strong>, para guardar minha Key. Tornado inacessível para usuários.
</p>
